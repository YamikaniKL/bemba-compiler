# bembajs-pisha

Vite-based build tool integration for BembaJS projects.

## Install

```bash
bun add bembajs-pisha
```

## Commands

```bash
pisha dev
pisha build
pisha preview
```

## Notes

- Uses `bembajs-core` compiler APIs for `.bemba` transforms.
- Compatible with strict package `exports` and modern ESM/CJS interop.
