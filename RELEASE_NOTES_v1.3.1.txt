BembaJS v1.3.1 Release Notes
============================

Release date: 2026-04-12

Packages released:
- bembajs@1.3.1
- bembajs-core@1.3.1
- create-bembajs@1.3.1

Highlights
----------
1. Compiler pipeline is now the default path in bembajs-core.
   - compile() now runs tokenize -> parse -> transform -> generate.
   - Legacy syntax fallback is still available for compatibility.

2. Parser and lexer reliability improvements.
   - Fixed lexer token issues that dropped first identifier/number characters.
   - Added missing operator tokens and parser alignment with token types.
   - Added tolerant parsing for complex component/object syntax patterns.

3. SDK and scaffolding consistency updates.
   - Updated versions/dependencies across bembajs, bembajs-core, create-bembajs.
   - Updated package READMEs with new compile() result behavior.
   - Added/fixed bembajs-core build script and generated dist shims.

Migration note
--------------
- compile() now returns an object with success/code fields on the main path.
- If your integration expected a direct HTML/string return, update to use:
  if (result.success) { result.code }
